# Github Stats Badges

**Repositório oficial e créditos:** [Repositório](https://github.com/anuraghazra/github-readme-stats)
**Site:** [Github stats](https://gh-stats-gen.vercel.app)

Github Stats permite que você mostre informações gerais sobre seu perfil, como número de commits, linguagens e informações sobre repositórios. Além disso, o site permite escolher diferentes temas/cores para o seu emblema.

## Como criar o seu

1. Entre no [site](https://gh-stats-gen.vercel.app): https://gh-stats-gen.vercel.app
<img width="834" height="456" alt="image" src="https://github.com/user-attachments/assets/a6fac1e0-55ae-4b9c-9b6c-c1c0f197d2a9" />

2. No campo "Github username" adicione o seu nome de usuário do github

3. Clique no campo theme para ver qual cor combina mais com seu perfil! Abaixo é possível ver a "preview" das cores escolhidas. 
<img width="503" height="267" alt="image" src="https://github.com/user-attachments/assets/c2f84028-2df3-44e4-bea5-dc405f56bdf8" />

4. Depois é só clicar em Show Result e clicar em cima de qual dos quadros você quer adicionar em seu Read Me
<img width="492" height="234" alt="image" src="https://github.com/user-attachments/assets/961aa958-f5f9-4b8e-a161-f517e699a47a" />

** Agora é só colar no seu perfil o que foi copiado **

O código abaixo em markdown:
```![Daniyutak's Stats](https://github-readme-stats.vercel.app/api?username=Daniyutak&theme=prussian&show_icons=true&hide_border=true&count_private=true)```

É mostrado como:
![Daniyutak's Stats](https://github-readme-stats.vercel.app/api?username=Daniyutak&theme=prussian&show_icons=true&hide_border=true&count_private=true)

## Usando html

É possível adicionar no html para dar display de imagem ou picture. Assim, também pode-se definir altura, largura e outras características do html.
É só colocar o html abaixo, colocando entre as aspas de src o link do seu stats

```
<img height="180px" alt="stats" src="https://github-readme-stats.vercel.app/api?username=SEUUSERNAME&theme=prussian&show_icons=true&hide_border=true&count_private=true">
```


Usando picture e sourceset é possível modificar um stats para versão dark mode/light mode do github de quem está vendo seu perfil. Essa feature pode ser entendida no arquivo light/dark mode desse repositório.


> O githubstats parou de funcionar algumas vezes, então recomendo ter salva uma versão do read me sem o github stats por precaução. Talvez haja formas de usar mesmo com ele caido, localmente ou algo do tipo. Consulte o (Repo oficial)[https://github.com/anuraghazra/github-readme-stats]
